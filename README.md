Source Code is 'main.cpp' in the new branch

To run in an IDE:
Compile and Run the Program
Input a command from the list of available commands:
"help" displays the commands and its description
"initialize" initialize the processor configuration
"exit" terminates the console
"screen -s <process name>" creates a new process
"screen -r <process name>" re=attaches to a running process
"screen -ls: lists all running processes
"scheduler-start" start the scheduler which continously generates a batch of dummy process for the CPU scheduler
"scheduler-stop" stop the scheduler
"report-util" generate the CPU utlization report

To run in the command in the command prompt:
Input 'g++ main.cpp -o main.exe'
Input 'main.exe'
